## 🎓 APRENDER O CÓDIGO

### Estrutura Lógica
1. **App.js** → Componente raiz
2. **Dashboard.js** → Gerencia estado global
3. **Components** → Recebem props e callbacks
4. **Services** → Chamam API

### Fluxo de Dados
```
User Input
    ↓
Component handler
    ↓
Dashboard function
    ↓
Service call
    ↓
API/Mock response
    ↓
State update
    ↓
Re-render
```

### Adicionar Feature
1. Decida onde vai
2. Adicione função em Dashboard
3. Passe como prop
4. Component chama em evento
5. State atualiza
6. UI re-renderiza
